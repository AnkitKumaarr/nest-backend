# Prisma Collections

This document lists all the collections (models) defined in the Prisma schema.

## Collections

- **User** - User accounts with authentication and profile information
- **Meeting** - Meeting schedules and details
- **ActivityLog** - User activity and notification logs
- **Company** - Company/organization information
- **CompanyUser** - Users associated with companies
- **Role** - User roles with permissions
- **Team** - Team/department information
- **TeamMember** - Members within teams
- **Project** - Project information
- **ProjectTask** - Tasks within projects
- **Comment** - Comments on tasks
- **Week** - Weekly calendar data
- **WeekTask** - Tasks scheduled within weeks
- **TaskPriority** - Task priority definitions
- **TaskStatus** - Task status definitions
- **TaskSnapshot** - Task analytics snapshots
- **MeetingSnapshot** - Meeting analytics snapshots
- **AnalyticsSnapshot** - General analytics data
- **Setting** - User settings and preferences
- **DeviceSession** - User device sessions
- **FileManager** - File management records

## Types (Not Collections)

The following types are defined but are not separate collections:
- Reply
- CreatedBy
- WeekDay
- WeekSlot
- WeekMonth
- DayDetails
